In this exercise, you'll create a simple chatbot that can remember previous interactions using LangChain's memory
components. You'll implement conversation memory to make your chatbot maintain context throughout a conversation.

**Instructions:**

3. Create a conversation chain with memory capabilities.
4. Implement a simple interactive chat interface.
5. Test the memory capabilities with a series of related questions.
6. Examine how the conversation history is stored and accessed.
